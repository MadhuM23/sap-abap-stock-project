# sap-abap-stock-project
# SAP ABAP Stock Management Reports

## Project Overview
This project contains SAP ABAP programs developed to manage
material and stock-related operations using different reporting
techniques in SAP.

## Programs Included
1. ZADD_MATERIAL  
   Program to add material master details.

2. ZADD_STOCK_MOV  
   Program to perform stock movement transactions.

3. ZSTOCK_CLASSICAL  
   Classical ABAP report to display stock details.

4. ZSTOCK_INTERACTIVE  
   Interactive ABAP report using events like AT LINE-SELECTION.

5. ZSTOCK_ALV  
   ALV report to display stock information in a structured format.

## ABAP Concepts Used
- Parameters and Select-options
- Internal Tables and Work Areas
- Modularization (FORM routines)
- Classical Reports
- Interactive Reports
- ALV Reports

## Tools & Environment
- SAP ABAP
- SAP GUI
- Transaction SE38

## Purpose
This project was developed for learning and practice purposes
to understand different ABAP reporting techniques and concepts.
